# MCI-Projekt

Dieses Repository enthält ein Projekt aus dem Bereich
Mensch-Computer-Interaktion (MCI).

## Ziel des Projekts

Ziel ist die Entwicklung einer interaktiven Desktopanwendung zur Darstellung
und Untersuchung verschiedener menschlicher Interaktions- und
Wahrnehmungsprozesse.

## Geplante Funktionen

- Visualisierung verschiedener Augenbewegungen
- Durchführung einer Aufgabe nach Fitts' Law
- Erfassung und Auswertung von Versuchsdaten
- Grafische Darstellung der Ergebnisse

## Technologien

- Python
- Tkinter

## Projektstatus

Das Projekt befindet sich derzeit in Entwicklung. Funktionen, Dokumentation
und Tests werden schrittweise ergänzt. Eine ausführliche Beschreibung der
Installation und Nutzung folgt zu einem späteren Zeitpunkt.

Interactive MCI experimente implementiert als GUI in Python 

# Überblick

Dieses Projekt wurde freiwillig im Rahmen des Moduls Mensch-Computer-Interaktion (MCI) an der Universität entwickelt.

Die Anwendung beinhaltet verschiedene interaktive Experimente und Visualisierungen zur Untersuchung von Benutzerinteraktionen, Wahrnehmung, Augenbewegungen und Gedächtnisleistungen. Ziel ist es, theoretische Konzepte der Mensch-Computer-Interaktion praktisch umzusetzen und die erhobenen Daten auszuwerten.

# Funktionen
**Augenbewegungen**
*     Simulation von Sakkaden
*     Simulation von Vergenzbewegungen
*     Simulation von Smooth-Pursuit-Bewegungen

# Programm starten
Man kann die Repository installieren oder in einem lokalen Ordner klonen durch
````git clone https://github.com/HamzaAhmed017/MCI-Project.git ````. 
Führe dannach den folgenden Befehl aus um die Abhängigkeiten zu installieren
````pip install -r requirements.txt````.
Als Letztes starte die main.py Datei durch ````python main.py```` im Ordner
